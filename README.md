# GitHub to GitLab auto-backup

## Summary

- Node.js v10 or later
- [GitHub REST API v3](https://developer.github.com/v3/)
- GitLab CE

## Usage

### Install

Just run `npm install`.

```bash
npm install
```

### Setup .env

```bash
cp .env.sample .env
vi .env
```

Set ALL variables in .env.

### Start backup

```bash
npm start
```
